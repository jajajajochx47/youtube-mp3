Python Local Tool

โหลด MP3 เป็นลิสต์แบบง่าย ๆ

วางลิงก์ YouTube หลายบรรทัด ระบบจะโหลดทีละรายการและเก็บชื่อเพลงเดิมให้มากที่สุด ใช้เฉพาะวิดีโอหรือเสียงที่คุณมีสิทธิ์ดาวน์โหลดเท่านั้น

Input

วางลิงก์ทีละบรรทัด

กำลังตรวจเครื่องมือ
ยังติดตั้งไม่ครบ

ต้องมี Python package yt-dlp และ ffmpeg ก่อนถึงจะโหลดเป็น MP3 ได้

python -m pip install yt-dlp
python -m pip install ffmpeg
